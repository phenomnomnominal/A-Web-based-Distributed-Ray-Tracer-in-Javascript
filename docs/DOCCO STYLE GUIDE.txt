### DOCCO STYLE GUIDE ###

Section Headings - 

# ___
# ## Title:

Section Headings with link - 

# ___
# ## <section id="title">Title:</section>

Sub-section Headings - 

# ___
# ### Title:

External Linked Documents/APIs - [**Titile**](link)

Strings - *'stringName'*
Objects - `objectName`
Constants, Classes and Functions - **`functionName`**

File names - *filename.fileext*

Additional information for listed items: # * LIST_ITEM - *Additional Information.*

Link to classes from different CoffeeScript files (once per class).
Link to functions from different classes (once per function).

Bullet points have no full stops at the end.
